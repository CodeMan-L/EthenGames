# 霓虹方块消消乐游戏文档

## 游戏信息
- **游戏ID**：Game002-blockpuzzle
- **游戏名称**：霓虹方块消消乐
- **英文名称**：Neon Block Puzzle
- **游戏类型**：益智、消除
- **图标**：🧩

## 游戏简介
霓虹方块消消乐是一款经典的方块消除游戏，玩家需要通过交换方块位置，使三个或以上相同颜色的方块连成一线进行消除。游戏采用霓虹风格的视觉效果，操作简单有趣，适合休闲时间玩耍。

```
Game002-blockpuzzle/
```

## 文件说明
- **index.html**：游戏的主页面，包含游戏的HTML结构
- **game.js**：游戏的核心逻辑，处理方块移动、消除等游戏机制
- **ui.js**：用户界面模块，处理游戏界面的显示和交互
- **audio.js**：音频处理模块，管理游戏音效
- **levels.js**：关卡配置文件，定义不同关卡的难度和布局
- **style.css**：样式文件，定义游戏的视觉效果

## 游戏特性
- **霓虹风格**：采用霓虹灯光效果，视觉效果炫酷
- **多种关卡**：包含多个不同难度的关卡
- **连击系统**：连续消除方块可以获得更高分数
- **音效反馈**：操作时有即时的音效反馈
- **响应式设计**：适配不同屏幕尺寸的设备

## 技术实现
- **前端技术**：HTML5 + CSS + JavaScript
- **事件处理**：使用鼠标和触摸事件实现游戏操作
- **动画效果**：使用CSS和JavaScript实现方块消除动画
- **音频**：集成音效增强游戏体验

## 游戏玩法
1. 打开游戏页面
2. 选择关卡难度
3. 通过点击或触摸交换相邻的方块位置
4. 当三个或以上相同颜色的方块连成一线时，方块会被消除
5. 消除足够数量的方块即可完成关卡
6. 挑战更高难度的关卡
