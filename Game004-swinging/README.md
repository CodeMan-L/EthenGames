# 霓虹秋千游戏文档

## 游戏信息
- **游戏ID**：Game004-swinging
- **游戏名称**：霓虹秋千
- **英文名称**：Neon Swinging
- **游戏类型**：休闲、动作
- **图标**：🎢

## 游戏简介
霓虹秋千是一款休闲动作游戏，玩家需要控制角色在秋千上摆动，通过点击屏幕来控制秋千的摆动节奏，使角色能够到达更远的距离。游戏采用霓虹风格的视觉效果，操作简单有趣，适合休闲时间玩耍。

## 文件结构
```
Game004-swinging/
├── audio.js            # 音频处理模块
├── game.js             # 游戏核心逻辑
├── index.html          # 游戏主页面
├── levels.js           # 关卡配置
├── style.css           # 样式文件
├── ui.js               # 用户界面模块
└── README.md           # 游戏文档
```

## 文件说明
- **index.html**：游戏的主页面，包含游戏的HTML结构
- **game.js**：游戏的核心逻辑，处理秋千摆动、角色移动等游戏机制
- **ui.js**：用户界面模块，处理游戏界面的显示和交互
- **audio.js**：音频处理模块，管理游戏音效
- **levels.js**：关卡配置文件，定义不同关卡的难度和布局
- **style.css**：样式文件，定义游戏的视觉效果

## 游戏特性
- **霓虹风格**：采用霓虹灯光效果，视觉效果炫酷
- **物理模拟**：真实的秋千摆动物理效果
- **多种关卡**：包含多个不同难度的关卡
- **音效反馈**：操作时有即时的音效反馈
- **响应式设计**：适配不同屏幕尺寸的设备

## 技术实现
- **前端技术**：HTML5 + CSS + JavaScript
- **物理引擎**：使用JavaScript实现简单的物理模拟
- **事件处理**：使用鼠标和触摸事件实现游戏操作
- **动画效果**：使用CSS和JavaScript实现角色和秋千的动画
- **音频**：集成音效增强游戏体验

## 游戏玩法
1. 打开游戏页面
2. 选择关卡难度
3. 点击屏幕来控制秋千的摆动节奏
4. 当秋千摆动到合适的角度时，松开屏幕使角色跳跃
5. 尽可能到达更远的距离
6. 挑战更高的分数
