# 粒子爆炸游戏文档

## 游戏信息
- **游戏ID**：Game009-particleblast
- **游戏名称**：粒子爆炸
- **英文名称**：Particle Blast
- **游戏类型**：休闲、动作
- **图标**：💥

## 游戏简介
粒子爆炸是一款以粒子效果为主题的休闲动作游戏，玩家需要通过点击屏幕来触发粒子爆炸，消除屏幕上的粒子。游戏采用霓虹风格的视觉效果，操作简单有趣，适合休闲时间玩耍。

## 文件说明
- **index.html**：游戏的主页面，包含游戏的HTML结构和基本布局
- **game.js**：游戏的核心逻辑，处理粒子生成、爆炸效果等游戏机制
- **specs/GUARDRAILS.md**：项目的边界守卫规范，定义了开发过程中的约束和规则
- **specs/PROJECT-CONTEXT.md**：项目上下文协议，描述了项目的背景和目标

## 游戏特性
- **霓虹风格**：采用霓虹灯光效果，视觉效果炫酷
- **粒子效果**：华丽的粒子爆炸效果
- **简单操作**：只需点击屏幕即可触发爆炸
- **音效反馈**：操作时有即时的音效反馈
- **响应式设计**：适配不同屏幕尺寸的设备

## 技术实现
- **前端技术**：HTML5 + CSS + JavaScript
- **Canvas**：使用Canvas绘制游戏画面和粒子效果
- **事件处理**：使用鼠标和触摸事件实现游戏操作
- **音频**：集成音效增强游戏体验

## 游戏玩法
1. 打开游戏页面
2. 点击屏幕任意位置触发粒子爆炸
3. 爆炸会消除周围的粒子
4. 尽可能消除更多的粒子
5. 挑战更高的分数
